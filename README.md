# KGL JavaScript Coursework

This repository contains three JavaScript coursework assignments for the Karibu Groceries LTD (KGL) data validation, inventory processing, and analytics system.

## Files

### 1. kgl_validation.js

**Coursework 1: KGL Data Validation System** (100 marks)

Topics covered:

- Variables, data types, and type checking
- String manipulation and validation
- Boolean operators and conditional logic
- Array operations and methods

### 2. kgl_functions.js

**Coursework 2: KGL Inventory Processing System** (100 marks)

Topics covered:

- Function declarations and arrow functions
- Object creation and manipulation
- Loops (for, for...of, for...in)
- Date objects and control flow

### 3. kgl_analytics.js

**Coursework 3: KGL Analytics and Reporting System** (100 marks)

Topics covered:

- Higher-order functions (map, filter, reduce)
- Sets for unique data management
- Maps for key-value storage
- Data transformation and analysis

## How to Run

Make sure you have Node.js installed on your system.

Run each file individually using Node.js:

```bash
node kgl_validation.js
node kgl_functions.js
node kgl_analytics.js
```

## Output

Each file will log results to the console, demonstrating:

- Variable declarations and type checking
- Data validation and transformation
- Business rule implementation
- Array and object manipulation
- Statistical calculations and reporting
